# Architecture

This folder describes the structural design of `newxos`.

## Observations

- [fact] `flake.nix` is generated by `flake-file` from repo declarations
- [decision] Module organization uses dendritic pattern with feature directories
- [requirement] Do not edit generated files directly when the project has a generator workflow
- [fact] `import-tree` recursively imports the `modules/` tree for module discovery
